# Portfolio-Angular

An Angular-based re-creation of my personal portfolio, originally built with React. This project is a **learning and experimentation project** aimed at understanding how to structure and build applications in Angular, while leveraging TypeScript, NodeJS, and standard web technologies.

## Key Learning Goals
- Understanding Angular architecture and component-based development
- Working with TypeScript in an Angular context
- Implementing routing and state management in Angular
- Experimenting with NodeJS integration and frontend-backend communication
- Translating a working React app into Angular to deepen framework understanding
